# 서울 생활환경 데이터를 활용한 범죄 위험도 예측 서비스
> 서울의 생활환경 데이터를 분석하여 지역별 범죄 위험도를 예측하고, 안전한 지역 추천해주는 서비스

---
# 주요 기능
- **범죄 위험도 예측**: 데이터 기반 지역별 범죄 발생 위험도 예측 및 지도 시각화
- **안전 지역 추천**: 안전 지역 TopN 추천
- **동네 비교**: 특징 비교 가능한 UI
- 생활환경 시각화: 범죄 위험도를 지도, 차트 형태로 제공 (규모를 키운다면, 시설 밀집도, 인구 밀집도 탭 기능 추가 고려)
---
## 수집 데이터

| 데이터 분류 | 데이터 설명 | 비고 |
|---|---|---|
| 치안 데이터(y) | 범죄발생정보 | 공간데이터마켓 |
| 안전 데이터 | 안전시설물 개수 | |
| 시설 데이터 | 학교, 공원, 병원, 유흥업소 위치 등 | 서울 열린데이터 광장 |
| 교통 데이터 | 지하철역, 버스정류장 위치 등 |
| 지형 데이터 | 경사도, 고도 등 | 중요도 떨어짐 |
| 인구 데이터 | 거주인구, 생활인구, 1인가구 데이터 등 |

## 기술

| 분야 | 기술 |
|---|---|
| 분석 모델 | Sckit-learn 기반의 회귀분석/ 분류분석 |
| 프론트엔드 | Streamlit |
| 지도, 시각화 | Folium, OpenStreetMap |
| 데이터 검증 | Great Expectations |

## AI 활용 방안

- 과거 범죄 발생 정보(y)와 인구, 교통, 시설, CCTV 유무 등 간의 관계를 학습해 특정 지역의 범죄 발생 위험도, 가능성 예측
- 분석 결과를 바탕으로 지역을 위험도 수준에 따라 분류, 군집화 

## 서비스 제공 방식

| 분야 | 제공 방식 |
|---|---|
| Backend | Go 언어 기반 API 수집 및 서버 구축 |
| Frontend | Streamlit 기반 웹 대시보드를 통해 범죄 위험도, 정보, 분석 결과를 지도, 차트 등의 형태로 시각화 |
| 지도 서비스 | OpenStreetMap 기반 지도를 활용해 데이터 직관적 제공 |

# 서울시 대기질 기반 야외운동 시간대 추천 서비스

> 빅데이터 분석 프로젝트 제안서

---

## 1. 서비스 주제

사용자가 자치구·운동 종류·희망 날짜를 입력하면, 향후 24시간의 PM10·PM2.5·오존 농도를 예측하고, 운동 강도에 따른 호흡량 차이를 반영해 시간대별 야외활동 적합도를 신호등(녹색 / 노랑 / 빨강)으로 제시한다. 부적합 시간대에는 같은 날 안에서 가장 권장되는 대체 시간대를 추천한다.

## 2. 핵심 차별점

단순히 "오늘 미세먼지 나쁨"을 보여주는 기존 앱과 달리 두 가지를 더한다.

1. **미래 시점 농도 예측** — "몇 시에 운동하라"는 행동 가능한 정보를 제공한다.
2. **운동 종류별 위험도 차등화** — 달리기는 분당 환기량이 안정 시의 10배 이상이라, 같은 농도라도 산책보다 흡입량이 훨씬 많다. 이 환기량 차이를 상대적 보정값으로 적용해 운동별로 다른 기준선을 적용하는 것이 핵심 아이디어다.

## 3. 구성별 상세

### 3.1 프론트엔드

- 입력 폼: 자치구 선택(드롭다운), 운동 종류 선택(걷기·조깅·자전거·등산·구기운동 등), 날짜 선택
- 결과 화면:
  - 선택한 날의 시간대별 적합도를 신호등 색상의 타임라인 막대로 표시
  - 예측 PM 농도 그래프
  - 추천 시간대 한 줄 요약
  - 운동 종류별 권장 지속시간 안내

### 3.2 서버(백엔드)

- 사용자 요청 수신 및 입력값(지역·운동·날짜) 검증
- AI 모델 추론 서버에 API 요청 → 시간대별 농도 예측값 수신
- 운동 종류별 호흡량 가중치를 적용해 적합도 등급을 계산하는 비즈니스 로직 수행
- 최종 추천 결과 및 사용자 요청 내역을 DB에 저장
- REST API 형태로 설계, 자주 조회되는 지역·날짜 조합은 캐싱해 응답 속도 향상

### 3.3 데이터

- **수집**: 서울시 대기질 측정소의 시간별 PM10·PM2.5·O₃·NO₂ 데이터 + 기상청의 기온·습도·풍속·풍향·강수 데이터
- **결합**: 측정소 위치 기준으로 대기질·기상 데이터를 시간·공간 결합, 자치구별 인접 측정소 매핑
- **전처리**:
  - 센서 결측·이상치(음수값, 급격한 튐) 제거
  - 시계열 보간(선형 보간 또는 전후 평균)으로 누락값 채우기
  - 시간(시·요일·계절)·기상 파생변수 생성
  - 정규화

### 3.4 AI 모델

- 과거 농도·기상 데이터를 입력으로 미래 시점의 PM10·PM2.5 농도를 예측하는 **회귀 모델** 학습
- 베이스라인: 선형 회귀
- 본 모델: 트리 기반 앙상블(Random Forest, XGBoost, LightGBM 등) 비교
- 시계열 특성 반영: 과거 N시간 농도를 피처로 넣는 lag feature 방식, 여력 시 LSTM 등 시계열 모델로 확장
- 평가지표: RMSE · MAE
- 하이퍼파라미터 튜닝(grid / random search)으로 성능 개선
- 농도 예측값을 환경부 대기환경 등급 기준에 맞춰 적합도 등급으로 변환

## 4. 활용 데이터셋

- 서울 열린데이터 광장 — 대기환경 측정정보
- 에어코리아 — 시간별 대기질 데이터
- 기상청 기상자료개방포털 — ASOS / AWS 관측자료

모두 공개되어 있어 데이터 확보가 안정적이다.

## 5. 기대 효과

천식·호흡기 질환자, 야외 운동을 즐기는 사람, 어린이 야외활동을 관리하는 보호자에게 "안전한 운동 시간"이라는 구체적 의사결정 정보를 제공한다.
